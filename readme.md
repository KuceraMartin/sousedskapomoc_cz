SousedskaPomoc.cz
===================

Webova stranka slouzici k organizaci dobrovolniku pro dorucovani zbozi lidem kteri si na nakup nemohou nebo nesmeji dojit.

Jakakoliv pomoc je vitana - cela aplikace bude uvolnena pod licenci GNU/GPL.
